สามารถ Run Application ผ่าน command line ได้เลยครับ 
โดย 
    - เข้ามาที่ Directory ./
    - พิมพ์ docker-compose up 
    - website จะอยู่ที่ port 3000
    - api จะอยู่ที่ port 5000
    - database อยู่ที่ database:27017/moomall 
    - database username = admin database password = 1234 
    - host = database ชื่อ database = moomall ชื่อ collection = products กับ orders
